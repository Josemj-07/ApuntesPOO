# 📘 Apuntes de Programación Orientada a Objetos en Java

Este repositorio contiene mis **apuntes de Programación Orientada a Objetos (POO)** aplicados al lenguaje **Java**, recopilados durante mi proceso de aprendizaje.

El objetivo de este material es que pueda servir como **guía de estudio**, **refuerzo conceptual** o **referencia rápida** para quienes estén aprendiendo POO o quieran aclarar conceptos que a veces resultan difíciles de entender al inicio.

---

## 🎯 Objetivo del repositorio

* Compartir mis **apuntes personales en formato PDF**.
* Ayudar a otras personas a **entender mejor los conceptos de POO**.
* Documentar mi **proceso de aprendizaje en Java**.
* Crear un recurso que pueda servir como **material de repaso**.

---

## 📄 Contenido

El repositorio incluye:

* 📘 **Apuntes completos de POO en Java (PDF)**
* Explicaciones ténicas con ejemplos intuitivos de conceptos clave

Algunos de los temas que se abordan:

* Clases y objetos
* Encapsulamiento
* Herencia
* Polimorfismo
* Abstracción
* Interfaces
* Constructores
* Métodos y atributos
* Ciclo de vida de los objetos
* Enums
* Records
* Buenas prácticas básicas en Java
* Entre otros...
---

## 📥 Cómo usar estos apuntes

1. Descarga el archivo PDF desde el repositorio.
2. Léelo como material de estudio o apoyo mientras aprendes Java.
3. Úsalo para repasar conceptos de POO cuando los necesites.
